# KMR-BEAUTY Logo Specifications

## Logo Principal

Le logo KMR-BEAUTY combine:
- **Texte "KMR"**: Police Poppins Bold, couleur blanche ou noire selon le fond
- **Illustration féminine**: Silhouette de femme avec cheveux stylisés, couleur Coral (#FF6B6B)
- **Texte "Beauty"**: Police script/cursive, couleur Coral (#FF6B6B)

## Variations du Logo

### 1. Logo Complet (avec illustration)
- **Usage**: Splash screen, onboarding, headers principaux
- **Fichier**: `logo_complete.svg` / `logo_complete.png`
- **Dimensions recommandées**: 200px x 80px minimum

### 2. Logo Texte Simple
- **Format**: "KMR-BEAUTY" en une ligne
- **Police**: Poppins Bold
- **Couleur**: #2D2D2D (sur fond clair) ou #FFFFFF (sur fond sombre)
- **Usage**: Navigation bar, headers compacts
- **Accent**: Ligne rouge/coral (#FF6B6B) sous "BEAUTY"

### 3. Logo Icône (App Icon)
- **Format**: Carré avec coins arrondis
- **Contenu**: Logo simplifié ou monogramme "K" avec illustration
- **Dimensions**: 1024x1024px (master), scales automatiques
- **Background**: Couleur coral ou gradient

### 4. Logo Footer / Inversé
- **Couleur**: Blanc sur fond sombre (#1A1A1A)
- **Usage**: Footer d'app, écrans dark mode
